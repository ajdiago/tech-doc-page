Bootstrap Quick Reference Guide Intro

Bootstrap makes building websites easy. The most popular framework for creating beautiful websites with a mobile first outlook. Easy to use compoenents and readily available documentation make it a breeze to learn and use. This QRG provides reference to some frequently used basics from the wide array of Bootstrap features.

Containers

The basis of layouts in Bootstrap. Containers control how sections of a website look and where they are positioned on the page. You can nest containers to be more specific in placement. There are three main container types:

  1. Base Container: sets a max width at each breakpoint
  2. Fluid Container: always 100% width
  3. Breakpoint Container: 100% until a specified breakpoint

Common responsive container examples: 

<div class="container-sm">100% wide until small breakpoint</div>
<div class="container-md">100% wide until medium breakpoint</div>
<div class="container-lg">100% wide until large breakpoint</div>
<div class="container-xl">100% wide until extra large breakpoint</div>
