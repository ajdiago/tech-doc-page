Bootstrap Quick Reference Guide Intro

Bootstrap makes building websites easy. The most popular framework for creating beautiful websites with a mobile first outlook. Easy to use compoenents and readily available documentation make it a breeze to learn and use. This QRG provides reference to some frequently used basics from the wide array of Bootstrap features.

Containers

The basis of layouts in Bootstrap. Containers control how sections of a website look and where they are positioned on the page. You can nest containers to be more specific in placement. There are three main container types:

  1. Base Container: sets a max width at each breakpoint
  2. Fluid Container: always 100% width
  3. Breakpoint Container: 100% until a specified breakpoint

Common responsive container examples: 

<div class="container-sm">100% wide until small breakpoint</div>
<div class="container-md">100% wide until medium breakpoint</div>
<div class="container-lg">100% wide until large breakpoint</div>
<div class="container-xl">100% wide until extra large breakpoint</div>
Cards

Cards are a useful container type that can contain their own header, footer, content, and background colors. They have unique display options. Cards are useful for product item listings, ads, and callout sections.

<div class="card" style="width: 18rem;"> <img src="..." class="card-img-top" alt="..."> <div class="card-body"> <h5 class="card-title">card title</h5> <p class="card-text">some quick example text to build on the card title and make up the bulk of the card's content.</p> <a href="#" class="btn btn-primary">go somewhere</a> </div> </div>

Buttons

Bootstrap offers a number of pre-built classes to create buttons. Buttons can be used with the btn, a, and input elements.

Navbars

Navbars are a useful tool to create fixed position or dynamic navigation tools for your page. Navbars can be horizontal or vertical. Placing a vertical navbar in a column will result in a sidebar style Navbar, like the one on this page.

  • Color Schemes

    <nav class="navbar navbar-dark bg-dark">
  <!-- navbar content -->
</nav>

<nav class="navbar navbar-dark bg-primary">
  <!-- navbar content -->
</nav>

<nav class="navbar navbar-light" style="background-color: #e3f2fd;">
  <!-- navbar content -->
</nav>

  • Sticky top

    <nav class="navbar sticky-top navbar-light bg-light">
  <a class="navbar-brand" href="#">sticky top</a>
</nav>
